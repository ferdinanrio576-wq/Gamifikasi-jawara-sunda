# Assets Folder

Folder ini berisi semua file statis untuk aplikasi Jawara Sunda.

## Struktur:
- `audio/` - File audio (musik latar, efek suara)
- `images/` - Gambar dan ikon
- `videos/` - File video jika diperlukan

## Cara Menambahkan File:
1. Letakkan file audio di `audio/` (format: .mp3, .wav, .ogg)
2. Letakkan gambar di `images/` (format: .png, .jpg, .svg)
3. Pastikan nama file tanpa spasi dan huruf kecil untuk konsistensi.

## Contoh Penggunaan:
- Musik latar: `assets/audio/background.mp3`
- Gambar avatar: `assets/images/avatar1.png`
